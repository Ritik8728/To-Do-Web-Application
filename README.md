# To-Do Web App

A modern and responsive To-Do Web Application built using HTML, CSS, and JavaScript. The application helps users manage daily tasks efficiently with features such as task creation, editing, deletion, filtering, sorting, priority management, and local storage support.

## Features

- Add, Edit and Delete Tasks
- Mark Tasks as Completed
- Task Priority Levels (Low, Medium, High)
- Task Categories
- Due Date and Time Support
- Search Tasks
- Filter Tasks (All, Active, Completed)
- Sort Tasks by Date, Priority, and Name
- Task Statistics Dashboard
- Local Storage Support
- Responsive Design for Mobile and Desktop

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Objective

The objective of this project is to develop a task management application that helps users organize, track, and manage their daily activities effectively.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `todo_app.html` file in any modern web browser.

## Key Functionalities

- Create and manage tasks
- Assign categories and priorities
- Set deadlines for tasks
- Search and filter tasks
- Track completed and pending tasks
- Store tasks using browser local storage
